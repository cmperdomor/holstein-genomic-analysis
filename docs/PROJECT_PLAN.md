# Holstein Genomic Analysis

## Objective

Characterize the genomic architecture and relationships among genomic evaluations in a commercial Holstein herd using Semex/Lactanet genomic data.

---

## Main Research Question

How can genomic evaluations be used to better understand the genetic structure of a commercial Holstein herd?

---

## Questions

- [ ] How has the genomic profile of the herd changed over time?

- [ ] How are genomic traits associated?

- [ ] Does Proof Source influence the observed relationships?

- [ ] Are there distinct genomic profiles among cows?

- [ ] Can cows be grouped using unsupervised learning?

- [ ] Which cows show the most balanced genomic profile?

- [ ] How has genomic inbreeding changed over time?

- [ ] Are there redundant genomic indexes?
